### semver
作用: 校验一个版本号是否合法semver.valid('1.2.3') // '1.2.3'。清除版本号里一些多余的语句semver.clean('  =v1.2.3   ') // '1.2.3'。比较版本号的大小semver.lt('1.2.3', '9.8.7') // true
